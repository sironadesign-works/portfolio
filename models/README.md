# Tiger I (WWII Germany) 3D Models

This folder contains two OBJ meshes inspired by the German Tiger I tank silhouette:

- `tiger_lowpoly.obj` — minimum low-poly blockout suitable for placeholders and prototyping.
- `tiger_highpoly.obj` — higher-density mesh with segmented wheels and barrel for closer viewing.

Both are generated by `../generate_tiger_models.py`.
